Beautiful Soup 库的使用

Beatiful Soup
Python 的一个 HTML 或 XML 的解析库

此库的一个很大的特点就是支持各种解析器

解析器

解析器 使用
Python标准库 BeautifulSoup(markup,“html.parser”)
lxml HTML BeautifulSoup(markup,“lxml”)
lxml XML BeautifulSoup(markup,“xml”)
html5lib BeautifulSoup(markup,“html5lib”)

初始化

通过 BeautifulSoup 将 str 类型的 html 文本转化成 “soup”

  • 实际上，在初始化 BeautifulSoup 时可以自动更正格式
  • soup.prettify() 可以把要解析的字符串以标准的缩进格式输出

节点选择器

  1. 直接调用节点名称->选择节点元素

提取信息

同上述 string, 如果需要提取节点的其他信息，比如 name， 同样只需要将 string 换成 name

获取：(同样引用上例)

  1. 名称：soup.p.name
  2. 属性：soup.p.attrs: 返回一个节点属性的字典，提取相应信息如：
    1. soup.p.attrs’name’
    2. soup.p.attrs[‘class’]：class返回的是一个列表，因为一个节点元素可能有多个 class
  3. 内容：soup.p.string

关联选择

选择时，有时不能做到一步就选到所需节点元素，需要先选中某一个节点元素，然后作为基准选择其父子节点、兄弟节点等
