XPath 的使用

XPath
XML Path Language

本文只简要介绍 Python 中 lxml 库的 etree 模块

XPath 常用规则

首先是 XPath 的常用规则：

表达式 描述
nodename 选取此节点的所有子节点
/ 从当前节点选取直接子节点
// 从当前节点选取子孙节点
. 选取当前节点
… 选取当前节点的父节点
@ 选取属性
* 匹配所有节点

除此以外，父节点也可通过 parent:: 来获取

etree 模块的基本用法

转化成 HTML 类

  1. HTML(text): 将字符串 text 初始化为 HTML 类，可自动修正HTML文本
  2. parse(’./test.html’,etree.HTMLParser()): 直接读取文本文件进行解析，成 HTML 类

HTML 类的使用——搜索节点

基本步骤

  1. 获得 HTML 对象
  2. 通过 HTML 类方法 xpath 进行解析： result=html.xpath(…)
    1. 其中 result 是一个列表（元素为节点，则类型为 Element）
    2. 一般会用 // 开头的 XPath 规则来选取所有符合要求的节点

实际上，如果希望最后得到的不是节点，而是节点的属性或是文本，同样可以在xpath()中体现：

  1. 属性: @attr，此处不加[]，否则表示属性匹配
  2. 文本: @text()

节点轴选择

xpath同样提供了很多节点轴的选取方法：

  1. ancestor:: 祖先节点
  2. attribute:: 属性值
  3. child:: 直接子节点
  4. descendant:: 所有子孙节点
  5. following:: 当前节点之后的所有节点
  6. following-sibling:: 当前节点之后的所有同级节点
