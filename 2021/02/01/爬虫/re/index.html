正则表达式 re 库

正则表达式并非 Python 独有，但 Python 的 re 库提供了整个正则表达式的实现，本文对此做简要介绍。

匹配规则

具体规则不赘述，只列举其中几点：

  1. w,s,d 的大小写对应含义相反，如\w,\W
  2. 贪婪&非贪婪
    1. 对于 pattern=…AB…
    2. 贪婪的情况：A会在B可满足的情况下尽可能多地匹配
    3. 字符串中间尽量使用非贪婪：.*?

基本方法

match & search

  1. match：从字符串开头匹配
  2. search：匹配时扫描整个字符串，返回第一个成功匹配结果

二者很相似：

  1. 参数格式相同: (pattern, content, 修饰符[1])
  2. 返回结果使用方法相同： result=func(…)
    1. group
      1. group(): 整个匹配字符串结果
      2. group(1): 第一个()提取的子字符串
    2. span()：返回匹配的范围

findall

参数格式同 match, search

返回结果的使用方式略有不同：

对于 result=findall(…)，返回的结果 result 实际上是一个列表，列表中每个元素是一个元组，元组中每个元素是提取的字符串——没有了匹配得到的整个字符串

sub

上述 3 种方法用来提取信息，而 sub 则可用来修改文本

参数中需要增加一个目标字符串=>(pattern, goal, content)

  • 将 content 中所有可用 pattern 匹配的部分替换成 goal

compile

值得注意的是前述参数中的 pattern 实际上并不是字符串，而应该是一个正则表达式对象

而 compile 则是一个用来将正则字符串编译成正则表达式对象的工具

  1. I:大小写不敏感, L:locale-aware, M:多行匹配, S:使.匹配包括换行在内的所有字符, U:根据Unicode字符集解析,X:给予更灵活的格式以便将正则表达式写得更易于理解 (e.g. re.I) ↩︎
