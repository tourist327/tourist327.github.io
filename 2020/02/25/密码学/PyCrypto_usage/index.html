流程解析

  • 对于加密与解密：
    • 从密钥文件中获取可实现encryption,decryption操作的cipher对象
      • key=f.read()
        • 获得string形式的RSA key
      • rsakey=RSA.importKey(key)
        • 将string转化成RSA key object
      • cipher=Cipher_pkcs1_v1_5.new(rsakey)
        • 得到可以用来实现PKCS#1 v1.5加密或解密的cipher object
    • 通过cipher对已知的明文或密文进行加密或解密
      • 加密
        • cipher.encrypt(message)
        • base64.b64encode(cipher.encrypt(message))
      • 解密
        • base64.b64decode(encrypt_text)
        • cipher.decrypt(base64.b64(encrypt_text),random_generator)
  • 对于签名与验签：
    • 从密钥文件中获取可实现signature,verification操作的signer对象或verifier对象
      • key=f.read()
      • rsakey=RSA.importKey(key)
      • 签名：signer=Signature_pkcs1_v1_5.new(rsakey)
      • 验签：verifier=Signature_pkcs1_v1_5.new(rsakey)
    • 对消息进行散列值处理（这里是SHA）
      • digest=SHA.new()
      • digest.update(message)
    • 签名
      • master通过自己的私钥产生的signer对digest（message取散列值）进行签名得到signature
      • sign=signer.sign(digest)
      • signature=base64.b64encode(sign)
    • 验签
      • ghost通过master的公钥产生的verify对收到的签名signature进行验证：自己计算digest，验证所收到signature是否与digest相对应
      • is_verify=verifier.verify(digest,base64.b64decode(signature))

关键函数

  • Crypto
  • Crypto.PublicKey.RSA
    • generate(bits, randfunc=None, progress_func=None, e=65537)
      • Randomly generate a fresh, new RSA key object.
      • return: An RSA key object
      • bits: Key length, or size (in bits) of the RSA modulus. It must be a multiple of 256, and no smaller than 1024.
      • randfunc: Random number generation function
    • importKey(externKey, passphrase=None)
      • string->_RSAobj
      • return: An RSA key object
      • externKey (string) - The RSA key to import, encoded as a string.
      • passphrase (string) - In case of an encrypted PEM key, this is the pass phrase from which the encryption key is derived.
    • construct(tup)
      • return: An RSA key object
      • tup (tuple) - A tuple of long integers, with at least 2 and no more than 6 items
      • n,e,d(PR),p(optional),q(optional),u((1/p)mod q, optional)
  • Crypto.Cipher.PKCS1_v1_5
    • new(key)
      • RSA key object->a cipher object
      • Return a cipher object PKCS115_Cipher that can be used to perform PKCS#1 v1.5 encryption or decryption.
  • Crypto.Signature.PKCS1_v1_5
    • new(key)
      • RSA key object->a signature scheme object
      • Return a signature scheme object PKCS115_SigScheme that can be used to perform PKCS#1 v1.5 signature or verification.
  • Crypto.Hash.SHA
    • new(data=None)
      • return: A SHA1Hash object
      • Return a fresh instance of the hash object.
  • Class SHA1Hash
  • Class _RSAobj
  • Class PKCS115_Cipher
  • Class PKCS115_SigScheme

签名

对于窃听者，有时候也可以对伪造Master给Ghost发送内容。为此出现了数字签名。也就是Master给Ghost发送消息的时候，先对消息进行签名，表明自己的身份，并且这个签名无法伪造。具体过程即Master使用自己的私钥对内容签名，然后Ghost使用Master的公钥进行验签。
